# Confirm-TerraformHashicorpKey

## SYNOPSIS
Helper function to sign the Hashi Corp PHP key.

## SYNTAX

```
Confirm-TerraformHashicorpKey [<CommonParameters>]
```

## DESCRIPTION
Helper function to sign the Hashi Corp PHP key.

## EXAMPLES

### EXAMPLE 1
```
Confirm-TerraformHashicorpKey
```

Runs gpg to sign a HasiCorp PGP key.
